O que é geofísica?

Geofísica é uma tecnociência que utiliza as propriedades físicas dos materiais (mais especificamente as rochas e minerais) para obter informações de subsuperfície. Normalmente busca-se: água subterrânea (aquíferos, aquitardes, aquífugos, etc.), depósitos minerais importantes para o país, petróleo e hidrocarbonetos relacionados, mapear zonas de contaminação do solo, dentre outros. Geofísicos utilizam metodologias indiretas, chamdas métodos, para obter as propriedades físicas do meio. Os nomes destes métodos normalmente, descrevem sua propriedade associada.

  • Método gravimétrico: utiliza a variação do campo gravitacional terrestre.
  • Método elétrico:fornece dados sobre os materiais a partir do estímulo elétrico dos materiais.
  • Método magnetotelúrico: fornece informações de subsuperfície a partir das correntes terrestres da ionosfera.
  • Método eletromagnético:utiliza a variação dos campos eletromagnéticoeticos terrestres.
  • Método radiométrico: mensura a radioatividade dos materiais.
  • Método magnético:fornece dados a partir da variação do campo magnético terrestre.
  • Método sísmico:fornece dados sobre os materiais de subsuperfície a partir de estímulos mecânicos. Assim como na imagem acima.
